# Pet Social FE

FrontEnd for Pet Social with Vuejs

## Feature

Similar Facebook but focus on Pet. Features: 

- auth
- profile
- post 
- chat

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run BackEnd

[pet-social-be](https://github.com/codegram01/pet-social-be)
